What's CocoPo

Twitter Lite( https://mobile.twitter.com/ )をElectronでラップしただけのTwitterクライアントです。

公式のをそのまま使っているので、投票などの機能も使えます。

そこまで軽量ではないと思いますが、見た目はすっきりさせることも可能です。

Download

CocoPo ver 1.3.2

今までのリリース

Theme

CocoPoはネット上に公開された任意のテーマをインストールする機能があります。

右上の歯車をクリックして設定画面を開き、URLを入力してInstallボタンをクリックするとテーマがインストールされます。

その後テーマを選んで変更すると、テーマが適用されます。

一度インストールに成功すると、アップデートボタン一つでテーマの更新が可能です。

CocoPo Dark

サンプルとしてCocoPoDarkと呼ばれる暗い色合いのテーマがあります。

https://lapistech.github.io/CocoPo/CocoPoDark/theme.json

テーマの公開については こちら をご覧ください。

Notes

フレームがない場合、左上に現れるアイコンをドラッグするとウィンドウの移動ができます。
